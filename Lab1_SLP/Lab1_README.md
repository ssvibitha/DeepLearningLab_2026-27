# Experiment 1 - Implementation of a Single Layer Perceptron for Binary Classification


This repository contains the implementation of **Single Layer Perceptron for Binary Classification**.

## Contents

- `Lab1.ipynb` – Google Colab compatible Jupyter Notebook
- `Lab1_Document.pdf` – Lab 1 Report
- `Lab1_Document.tex` - Lab 1 Report (Latex file)
- `plots.eps` - Plots and visualisation (.eps files)

## Objective

1. Implement a **Single Layer Perceptron from scratch** to classify the Banknote Authentication Dataset.

## Dataset

- **Dataset:** Banknote Authentication Dataset
- **Source:** UCI Machine Learning Repository

## Running the Notebook

Open the `.ipynb` file in **Google Colab** and run all cells sequentially. 

(No additional setup is required apart from installing any missing Python libraries if prompted.)

## Outputs

The notebook includes:

- Exploratory Data Analysis (EDA)
- Data preprocessing
- Perceptron implementation from scratch
- Model training
- Performance evaluation
- Required plots and visualizations
